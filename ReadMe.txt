COMP2406 A3

Tested on Google Chrome with Node 8.11.4 installed

To install:
Download the .zip submission.
Unzip to the folder of your choice
Using cmd, navigate to the folder you unzipped and then to "COMP2406F18A3"
Once cmd is in that directory, enter "node server.js" to start the server.

To run:
As per the server's instructions, go to http://localhost:3000/assignment3.html once it is running

To play:
As per the instructions, click the appropriate button to take/give up control of a rock. Click on a rock and drag to 'catapult' it.
